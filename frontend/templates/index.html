关于系统

系统简介

多模态谣言检测系统是一个基于深度学习的智能分析平台，融合了CLIP视觉-语言预训练模型、 BERT文本编码、VGG图像特征提取等技术，能够自动分析社交媒体内容中的文本和图像信息， 判断其真实性，为遏制虚假信息传播提供技术支持。

技术架构

  • 文本特征提取：BERT (chinese_L-12_H-768_A-12)
  • 图像特征提取：VGG-19
  • 跨模态特征：CLIP (ViT-B/32)
  • 分类模型：三层全连接神经网络 (1024→256→16→2)
  • 特征融合：基于余弦相似度的自适应加权融合

核心功能

  • ✅ 实时多模态谣言检测
  • ✅ 图文一致性分析
  • ✅ 可视化概率展示
  • ✅ API接口支持
  • ✅ 批量检测支持

API文档

系统提供的RESTful API接口说明

健康检查接口

GET /api/health

检查系统运行状态和模型加载情况

响应示例

{
  "status": "ok",
  "model_loaded": true,
  "device": "cuda",
  "timestamp": "2024-01-01T12:00:00"
}

单条预测接口

POST /api/predict

对单条图文内容进行谣言检测

请求体

{
  "text": "新闻文本内容...",
  "image": "data:image/jpeg;base64,..."
}

响应示例

{
  "success": true,
  "data": {
    "prediction": "谣言",
    "confidence": 0.85,
    "probabilities": {
      "非谣言": 0.15,
      "谣言": 0.85
    },
    "reasoning": "分析说明..."
  }
}

批量预测接口

POST /api/batch_predict

批量检测多条图文内容（最多10条）

请求体

{
  "items": [
    {"text": "文本1", "image": "base64_1"},
    {"text": "文本2", "image": "base64_2"}
  ]
}
